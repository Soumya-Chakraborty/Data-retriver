# Extension Icons

This directory contains the icons for the Data Entry Tool Firefox extension.

## Required Icons

The extension requires the following icon files:

- `icon16.png` - 16x16 pixel icon for the browser toolbar
- `icon48.png` - 48x48 pixel icon for extension details
- `icon128.png` - 128x128 pixel icon for extension management

## How to Obtain Icons

You can create or download icons from the following sources:

1. **Free Icon Resources:**
   - [Flaticon](https://www.flaticon.com/)
   - [Iconfinder](https://www.iconfinder.com/)
   - [Icons8](https://icons8.com/)
   - [Feather Icons](https://feathericons.com/)
   - [Heroicons](https://heroicons.com/)

2. **Create Your Own:**
   - Use an image editor like GIMP or Photoshop
   - Use online tools like Canva or Figma
   - Use vector tools like Inkscape

## Icon Suggestions

For a data entry tool, consider icons related to:
- Databases
- Forms/tables
- Pencils/pens
- Checklists
- Data/collection
- Clipboard
- Spreadsheet

## Icon Specifications

- Format: PNG (recommended) or SVG
- Background: Preferably transparent
- Style: Simple and recognizable even at small sizes
- Colors: Should contrast well with dark and light themes

## How to Add Icons

1. Download or create your icons in the required sizes
2. Save them as PNG files with the following names:
   - `icon16.png` (16x16 pixels)
   - `icon48.png` (48x48 pixels) 
   - `icon128.png` (128x128 pixels)
3. Place them in this directory
